DASH for Azure Storage

DASH is a solution from Microsoft that allows you to bypass space and I/O limits on Azure Storage. In order to use DASH, you will need to download the source, build the package, and deploy it to Azure.

DASH works by aggregating the throughput, volume and transaction limits of multiple Azure Blob Storage Accounts to create an effective virtual account that has limits of N times those imposed on a single storage account.

Internally, DASH maintains a lookup of all blob resources in the virtual account (the 'namespace' account). These are 0-length blobs that match the external structure of account. In each namespace blob, a small amount of metadata is held.
